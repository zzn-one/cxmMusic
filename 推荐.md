# 默认推荐的标签列表

- 根据歌单的标签分组，统计歌单的播放量与收藏量

  ```sql
  SELECT
  	tag,
  	SUM( star_number ) sum_star,
  	SUM( play_number ) sum_play 
  FROM
  	songlist 
  WHERE
  	deleted = 1
      and 
      tag!=''
  GROUP BY
  	tag
  ```

  结果实体类

  ```java
  /**
   *
   * 根据标签分组 统计歌单的播放量和收藏量
   * @create 2023-04-12 8:58
   */
  @Data
  @AllArgsConstructor
  @NoArgsConstructor
  public class SumStarNumberAndPlayNumber {
  
      private String tag;
  
      private Long sumStar;
  
      private Long sumPlay;
  
      private Long sumTotal;
  
      /*
      * 计算总和 播放量权重为1 收藏量权重为2
      * */
      public void total(){
          this.sumTotal = this.sumPlay + this.sumStar * 2L;
      }
  }
  
  ```
